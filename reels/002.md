## 延伸學習

### 核心觀點
- Prompt Engineering 本質上是「把需求講清楚」的能力。
- 好的 Prompt = 明確的角色 + 具體的任務 + 清晰的格式要求。

### Prompt 框架速查
| 元素 | 說明 | 範例 |
|------|------|------|
| Role | 指定 AI 的角色 | 你是一位資深行銷顧問 |
| Task | 明確的任務描述 | 幫我撰寫一篇產品介紹 |
| Format | 期望的輸出格式 | 用條列式，不超過 300 字 |
| Context | 背景資訊 | 目標受眾是 25-35 歲上班族 |

### 思考題
1. 試著用 Role-Task-Format 框架重新改寫你上一次對 ChatGPT 的提問。
2. 比較改寫前後的回覆品質差異。
